# 🍽️ YumYumCoach

식단/운동 기록, 챌린지, 커뮤니티, AI 분석/추천을 제공하는 헬스케어 서비스입니다. 각 영역은 별도 레포지토리로 운영됩니다.

## ✨ 서비스 목표
바쁜 일상에서 식단과 운동을 꾸준히 기록하고 평가받는 것이 어렵고, 동기 부여가 쉽게 떨어질 수 있습니다.<br> 
YumYumCoach는 기록-분석-피드백-동기부여를 하나의 흐름으로 연결해 건강한 습관 형성을 돕는 것을 목표로 합니다.

## 레포지토리

| Area | Description | Link |
|------|-------------|------|
| **Frontend** | 사용자 UI 및 클라이언트 로직 (Vue 3, Vite) | [Frontend](https://github.com/ssafy-yumyumcoach/frontend) |
| **Backend** | API 서버, 인증·비즈니스 로직 (Spring Boot) | [Backend](https://github.com/ssafy-yumyumcoach/backend) |
| **AI** | 이미지 음식 분석 모델 서버 (FastAPI, YOLO) | [AI](https://github.com/ssafy-yumyumcoach/AI) |


## 제공 서비스
- 로그인/회원가입, 프로필/팔로우
- 음식 검색/식단 기록, 운동 검색/기록
- 커뮤니티 게시글/댓글/좋아요
- 챌린지 참여/진행
- 주간 통계 요약
- AI: 음식 이미지 감지, 식단/운동 평가, 식단 추천, 챗봇
